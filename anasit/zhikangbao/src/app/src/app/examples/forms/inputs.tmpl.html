Text fields allow the user to input text. They can be single line, with or without scrolling, or multi-line, and can have an icon.

Touching a text field places the cursor and automatically displays the keyboard. In addition to typing, text fields allow for a variety of other tasks, such as text selection (cut, copy, paste) and data lookup via auto-completion. See Patterns > Selection for text selection design.

Floating labels

With floating inline labels, when the user engages with the text input field, the labels move to float above the field.

Forms with icons

You can also add informative icons to your input fields.

Inputs with counter

You can use a character counter in fields where a character restriction is in place.

Input state

You can also toggle the state of inputs by setting ng-disabled
